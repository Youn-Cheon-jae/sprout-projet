Git Bash 통해서 이름.이메일 다시 입력 후

git init 입력 git add. 추가 파일 더하기 git status 상태확인 git commit -m"first commit" 만들기 git remote add origin https://github.com/chaunjan/sprout-projet.git git remote -v 연결상태 확인 git push origin master 계속 업데이트할때는 3개만 git add. git commit -m git push origin master
